# Secure Password Generator

A robust desktop utility designed to generate highly secure, customizable random passwords. This application focuses on algorithmic character selection, conditional logic, complex form handling, and user-friendly GUI state management.

## 🚀 Key Features

- **Bulk Generation:** Generate multiple unique passwords (from 1 to 10) in a single execution.
- **Flexible Length Control:** Supports customizable password lengths ranging from 5 to 20 characters.
- **Advanced Character Filtering:** Toggle specific character sets including digits, lowercase/uppercase English letters, and special symbols (`# % &`).
- **Ambiguous Characters Exclusion:** Built-in smart filter to exclude visually similar characters (like `i, l, 1, L, !, o, O, 0`) to prevent user confusion.
- **Batch Actions:** Quick "Select All" and "Clear All" features for efficient configuration management.

## 🛠 Tech Stack

- **Language:** Python 3.11+
- **GUI Framework:** CustomTkinter
- **Architecture:** Object-Oriented Programming (OOP) with dynamic form validation.

   ```bash

   ```
2. Install dependencies:
   ```bash

   ```

